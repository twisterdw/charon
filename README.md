рЃћрЃА рЃАрЃљрЃфрЃљрЃЋрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА Charon рЃЋрЃљрЃџрЃўрЃЊрЃўрЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃљрЃЊрЃў рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃгрЃДрЃљрЃарЃЮрЃА рЃЎрЃЮрЃЊрЃА (рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЦрЃЏрЃўрЃА "kharon"); HTTP рЃерЃБрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃЎрЃџрЃўрЃћрЃюрЃбрЃў Ethereum Staking-рЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ рЃћрЃарЃЌрЃў рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃў рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃЎрЃЋрЃљрЃюрЃФрЃћрЃЉрЃўрЃА рЃ»рЃњрЃБрЃцрЃќрЃћ.

Charon-рЃА рЃљрЃ«рЃџрЃљрЃЋрЃА рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃАрЃљрЃ«рЃћрЃџрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃЌ [Distributed Validator Launchpad](https://goerli.launchpad.obol.tech/) рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃљрЃЊрЃў рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃАрЃбрЃћрЃўрЃЎрЃћрЃарЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю charon-рЃА Ethereum рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃљрЃќрЃћ рЃЊрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃќрЃћ.

###### рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃљрЃЊрЃў рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃўрЃА рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Charon рЃЎрЃџрЃўрЃћрЃюрЃбрЃА рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃўрЃА рЃБрЃЎрЃЏрЃљрЃарЃўрЃАрЃЮрЃЉрЃўрЃА рЃарЃўрЃАрЃЎрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃфрЃљрЃЋрЃљрЃЊ.

## рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

charon-рЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃБрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃАрЃў рЃњрЃќрЃљрЃљ[charon-distributed-validator-cluster](https://github.com/ObolNetwork/charon-distributed-validator-cluster) рЃарЃћрЃърЃЮ
рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЊрЃЮрЃЎрЃћрЃарЃўрЃА рЃерЃћрЃЊрЃњрЃћрЃюрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљрЃА, рЃарЃљрЃЌрЃљ рЃљрЃгрЃљрЃарЃЏрЃЮрЃЮрЃА рЃАрЃарЃБрЃџрЃў рЃЦрЃљрЃарЃЮрЃюрЃўрЃА рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋ рЃљрЃърЃљрЃарЃљрЃбрЃќрЃћ.

## рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

[Obol Docs](https://docs.obol.tech/) рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃў рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃљ рЃњрЃљрЃАрЃљрЃгрЃДрЃћрЃЉрЃљрЃЊ.
рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃАрЃћрЃЦрЃфрЃўрЃљ рЃљрЃарЃўрЃА [intro](https://docs.obol.tech/docs/intro),

## рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃџрЃў рЃЎрЃЮрЃюрЃАрЃћрЃюрЃАрЃБрЃАрЃўрЃА рЃцрЃћрЃюрЃўрЃА рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃў

Charon рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃЊрЃћрЃЉрЃљ Ethereum-рЃўрЃА рЃЎрЃЮрЃюрЃАрЃћрЃюрЃАрЃБрЃАрЃўрЃА рЃАрЃбрЃћрЃЎрЃерЃў, рЃарЃЮрЃњрЃЮрЃарЃф рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃа рЃЎрЃџрЃўрЃћрЃюрЃбрЃА рЃерЃЮрЃарЃўрЃА
рЃЊрЃљ рЃерЃБрЃЦрЃБрЃарЃљ рЃЎрЃЋрЃљрЃюрЃФрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў [Eth Beacon Node REST API](https://ethereum.github.io/beacon-APIs/#/).
рЃЦрЃљрЃарЃЮрЃюрЃў рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃќрЃћрЃЊрЃљ рЃЊрЃўрЃюрЃћрЃЉрЃўрЃА рЃерЃБрЃЦрЃБрЃарЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ Beacon API-рЃА.
Charon рЃЏрЃўрЃќрЃюрЃљрЃЊ рЃўрЃАрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃ«рЃљрЃарЃў рЃЊрЃљрЃБрЃГрЃўрЃарЃЮрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџ рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃа рЃЎрЃџрЃўрЃћрЃюрЃбрЃА Beacon API-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў

Obol Network-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃ»рЃћрЃа рЃљрЃЊрЃарЃћрЃљ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃўрЃљ.
рЃЕрЃЋрЃћрЃю рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЏрЃўрЃЋрЃўрЃгрЃћрЃЋрЃЌ рЃгрЃўрЃю, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃљрЃЊ рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃърЃарЃЮрЃњрЃарЃћрЃАрЃў.

рЃЦрЃљрЃарЃЮрЃюрЃў рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљрЃЊрЃў рЃЋрЃљрЃџрЃўрЃЊрЃљрЃбрЃЮрЃарЃў, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃЋрЃљрЃџрЃћрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ.
рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃфрЃ«рЃарЃўрЃџрЃў рЃљрЃАрЃљрЃ«рЃљрЃЋрЃА рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃћрЃџрЃў рЃЏрЃЮрЃЋрЃљрЃџрЃћрЃЮрЃЉрЃћрЃЉрЃў рЃерЃћрЃЦрЃЏрЃюрЃћрЃА рЃАрЃљрЃ»рЃљрЃарЃЮ рЃАрЃљрЃфрЃЊрЃћрЃџ рЃЦрЃАрЃћрЃџрЃќрЃћ рЃЊрЃљ рЃарЃЮрЃЏрЃћрЃџрЃў рЃ»рЃћрЃа рЃЎрЃўрЃЊрЃћрЃЋ рЃЏрЃерЃћрЃюрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃўрЃљ (­ЪџД )
